회원가입 시 API 키 발급 방법

YouTube API 키 발급 방법

    API 키의 예) AIzaSy.....
  1. Google Cloud Console에 접속합니다.
  2. 로그인합니다. (Google 계정이 없을 경우 계정을 생성해야 합니다.)
  3. 상단의 "프로젝트 선택"을 클릭한 후 뜨는 팝업에서 "새 프로젝트"를 선택합니다.
  4. 프로젝트 이름을 입력 후 만들기를 선택합니다. (프로젝트 이름은 자유입니다.)
  5. 프로젝트가 생성되면 "API 및 서비스"로 이동 후 "라이브러리"를 선택합니다.
  6. 검색창에 "YouTube Data API v3"를 입력하고, 검색 결과에서 API를 클릭합니다.
  7. "사용" 버튼을 클릭하여 YouTube Data API를 활성화합니다.
  8. "API 및 서비스" 메뉴에서 "사용자 인증 정보"로 이동합니다.
  9. "인증 정보 만들기"를 클릭한 후 "API 키"를 선택합니다.
  10. 생성된 API 키가 화면에 표시되며, 이 키를 복사해서 회원가입 창 YouTubeAPI 칸에 입력합니다.

Wiki API 키 발급 방법

  1. Wikifier에 접속합니다.
  2. "User name"에 당신의 이름을 입력합니다.
  3. "Register!"를 누릅니다.
  4. "Your user key is:" 다음에 있는 키를 회원가입 창 WikiAPI 칸에 입력합니다.

우리는 Bloom의 분류학을 사용하여 현재 학습 단계에 대해 알려드립니다.

Bloom의 분류법
